LineChart visualization

Purpose

The purpose of this visualization is to see which months are the most popular for travelling. This data can be used, for example, by airports, knowing which months are the most crowded in a year, the airport can hist some part-time stuff for handling bigger amount and reducing large lines.

Wrangling

The data set "Activity period" was grouped by each month for all years.

Tableau Prototype

Prototype was done using Tableau. The data used: "Passenger Number" (sum for each month group) (as y scale) and "Activity period" grouped by months (as x scale).

Visualization

Passenger number by months
 Data acquired from data.sfgov.org/ on February 8, 2020.
          Elizaveta Ozerova

The JavaScript code for this visualization can be found here.

Analysis

Passenger count is encoded as the y line, so higher points on graph represent higher passenger count. Activity period by months is encoded as the x line, so the points location represent the total amount of passengers at the particular month.

Using this visualization we can conclude that airports and planes have more passengers during in summer time and big drop in February. This data may help airport's administration to correct their plans on employees.
